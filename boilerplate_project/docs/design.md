# Tài liệu thiết kế

- Primary: #007bff (xanh dương)
- Secondary: #6c757d (xám)

- Danger: #dc3545 (đỏ)
- Warning: #ffc107 (vàng)

## Breakpoints
- Mobile: < 768px
- Tablet: 768px - 1024px

## Components
- Buttons: Rounded corners, hover effects

- Cards: Shadow effects, hover animations
- Navigation: Sticky header, mobile menu
